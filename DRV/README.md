# Применение дискретных распределений на примере различных задач

## Задача:
Рассмотреть применение биномиального, распределения Пуассона, геометрического и гипергеометрического распределений.

Задачи включают:
1) Биномиальное распределение: число кликов по рекламному баннеру из 20 показов при вероятности клика 0.15.
2) Распределение Пуассона: число автомобилей, проезжающих за минуту на перекрёстке в среднем 3 автомобиля.
3) Геометрическое распределение: число покупателей, которых нужно обслужить до того, как кто-то выберет бестселлер с вероятностью 0.15.
4) Гипергеометрическое распределение: число зелёных яблок среди 5 случайно выбранных из корзины с 30 яблоками, 6 из которых зелёные.

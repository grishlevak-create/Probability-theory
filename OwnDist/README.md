# Создание собственных распределений в Python

## Задача:
Реализовать собственное непрерывное распределение вероятностей в Python с использованием класса rv_continuous.
Задать функцию плотности вероятности, определить соответствующую функцию распределения и исследовать основные числовые характеристики случайной величины.

Для заданного распределения требуется:

1) проверить условие нормировки плотности;

2) вычислить значения функции распределения в граничных точках;

3) сгенерировать выборку случайных значений;

4) вычислить математическое ожидание, медиану, дисперсию и среднее квадратическое отклонение;

5) определить коэффициент асимметрии и эксцесс;

6) вычислить квантили заданных уровней.
